# 볼린저밴드 스캐너

## 지표 파라미터

| 항목 | 값 |
|------|-----|
| 기간 (Period) | 20 |
| 표준편차 배수 | 2.0σ |
| 기준 컬럼 | `close` |
| 라이브러리 | `pandas_ta` |

---

## 스캔 조건

| 상태 | 조건 | 표시 |
|------|------|------|
| 하단 터치 | `close ≤ BBL_20_2.0` | 🔵 하단 |
| 상단 터치 | `close ≥ BBU_20_2.0` | 🔴 상단 |
| 정상 | 그 외 | — |

최소 데이터 포인트: **20개** (밴드 계산 불가 시 스킵)

---

## 스캔 타임프레임

### 3분봉 스캔
- 데이터 소스: KIS `FHKST03010230` / AlphaVantage `TIME_SERIES_INTRADAY interval=3min`
- 조회 범위: 최근 100개 캔들 (≈ 5시간)
- 실행 시점: 수동 스캔 버튼 or 자동 반복 (N분 간격)

### 일봉 스캔
- 데이터 소스: KIS `FHKST03010100` / AlphaVantage `TIME_SERIES_DAILY_ADJUSTED`
- 조회 범위: 최근 60일 (볼린저밴드 안정화 충분)
- 실행 시점: 수동 스캔 버튼 or 앱 시작 시 1회

---

## 스캔 흐름

```
카테고리 선택 (또는 전체)
  │
  ├─ 카테고리 내 종목 목록 로드
  │
  ├─ 종목별 루프
  │     ├─ OHLCV 조회 (캐시 우선)
  │     ├─ 볼린저밴드 계산
  │     ├─ 마지막 캔들 close vs BBL/BBU 비교
  │     └─ 터치 시 결과 리스트에 추가
  │
  └─ 결과 패널에 표시 (최신순 정렬)
```

---

## 스캔 결과 UI 컬럼

| 컬럼 | 내용 |
|------|------|
| 종목명 | 한글명 또는 티커 |
| 시장 | KR / US |
| 카테고리 | 반도체, Technology 등 |
| 타임프레임 | 3분봉 / 일봉 |
| 상태 | 상단 터치 / 하단 터치 |
| 현재가 | 마지막 close 값 |
| BBL / BBU | 밴드 값 |
| 감지 시각 | HH:MM:SS |

---

## 성능 고려사항

- KIS Rate Limit: 종목당 0.2초 delay → 100종목 스캔 ≈ 20초
- AlphaVantage 무료: 캐시 없으면 25종목 한도
- 스캔 진행률 프로그레스바 표시 권장
