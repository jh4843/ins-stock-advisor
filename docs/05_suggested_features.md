# 추가 제안 기능

## 1. 스캔 자동 반복

설정 가능한 주기(3분, 5분, 10분)로 볼린저밴드 스캔 자동 실행.  
이전 결과와 비교해 **신규 터치 종목만 강조 표시**.

```
UI: "자동 스캔" 토글 + 주기 선택 콤보박스
새 결과: 배경색 하이라이트 + 3초 후 일반 색으로 페이드
```

---

## 2. OS 토스트 알림

볼린저밴드 터치 감지 시 OS 시스템 알림 + 알림음.

- Windows: `plyer` 라이브러리 (`pip install plyer`)
- 설정: 알림 ON/OFF, 상단/하단 구분 알림 ON/OFF
- 중복 방지: 동일 종목 동일 상태는 10분 내 재알림 없음

---

## 3. 종목 즐겨찾기 (Watchlist)

자주 보는 종목을 즐겨찾기 등록해 별도 탭에서 빠르게 접근.

```
저장: ~/.inz_stock_advisor/data/watchlist.json
UI: ★ 버튼으로 토글, 즐겨찾기 탭 별도 구성
```

---

## 4. 멀티타임프레임 스캔 결과 탭

스캔 결과 패널을 **3분봉 탭 / 일봉 탭**으로 분리.  
양쪽 모두 터치한 종목은 "강한 신호"로 별도 표시.

---

## 5. 수급 오버레이 (KR 종목)

차트 하단에 외국인/기관 순매수 막대 그래프 오버레이.

- 데이터: KIS `inquire-investor` API
- 표시: 외국인 (파랑), 기관 (빨강) 막대
- 토글 버튼으로 ON/OFF

---

## 6. 차트 추가 지표

현재 볼린저밴드 외에 선택적으로 표시할 수 있는 지표:

| 지표 | 라이브러리 | 용도 |
|------|-----------|------|
| RSI (14) | pandas_ta | 과매수/과매도 |
| MACD | pandas_ta | 추세 전환 |
| 거래량 MA | pandas | 거래량 이상 감지 |

---

## 7. 종목 검색

좌측 패널 상단에 검색창 추가. 코드 또는 이름 입력 시 실시간 필터링.

```
KR: all_stocks.csv 에서 검색
US: us_watchlist.json + OVERVIEW 캐시에서 검색
```

---

## 우선순위 제안

즉시 효과가 큰 기능 순:

1. **스캔 자동 반복** — 핵심 사용 시나리오
2. **멀티타임프레임 탭** — 3분봉/일봉 동시 모니터링
3. **OS 토스트 알림** — 화면 안 봐도 인지 가능
4. **종목 검색** — 카테고리 수 늘어날수록 필수
5. **수급 오버레이** — 매매 판단 보조
